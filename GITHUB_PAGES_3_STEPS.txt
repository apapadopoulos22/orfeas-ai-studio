TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════

❌ Problem: "Repository not found" error when pushing

✓ Solution:
  • Make sure you replaced YOUR-USERNAME with YOUR actual GitHub username
  • Check capitalization matters
  • Verify you created the repository on GitHub first
  • Try: git remote -v (to check the URL is correct)


❌ Problem: "fatal: could not read Password"

✓ Solution:
  • Use GitHub Personal Access Token instead of password
  • Create token: https://github.com/settings/tokens
  • Use token as password when prompted


❌ Problem: GitHub Pages not showing in Settings

✓ Solution:
  • Wait 5 minutes (GitHub needs time to process)
  • Refresh the page
  • Check "Actions" tab to see build status


❌ Problem: Site shows 404 error

✓ Solution:
  • GitHub is looking for index.html
  • Your main file is synexa-style-studio.html
  • Rename it to index.html, OR
  • Edit netlify.toml to point to synexa-style-studio.html
  • Push the change: git add . && git commit -m "Fix index" && git push


❌ Problem: API calls failing (can't generate 3D)

✓ Solution:
  • Start backend: python backend/main.py
  • Start ngrok: .\START_NGROK_TUNNEL.bat
  • Check ngrok shows: "Forwarding https://... -> http://localhost:5000"
  • Check F12 console for error messages
