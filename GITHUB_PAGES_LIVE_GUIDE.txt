TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════

❌ Still showing 404 after 5 minutes?

✓ Solution:
  1. Go to: https://github.com/apapadopoulos22/orfeas-ai-studio
  2. Click: "Actions" tab
  3. Check if build failed (red ❌)
  4. If failed: Click the failed action to see error
  5. Common cause: Large files in LFS (not blocking)
  6. Try: Hard refresh (Ctrl+Shift+R) in browser

❌ Page loads but shows blank/loading forever?

✓ Solution:
  1. Press F12 → Console tab
  2. Look for red error messages
  3. Common: API calls failing
     → Make sure backend is running
     → Make sure ngrok tunnel is active
  4. Check API_BASE in console matches ngrok URL

❌ Upload button doesn't work?

✓ Solution:
  1. Backend not running
     → Start: cd backend && python -u main.py
  2. ngrok not running
     → Start: .\START_NGROK_TUNNEL.bat
  3. Check backend is on http://127.0.0.1:5000/health
     → Should return "ok"

❌ Generation fails / "Cannot reach backend"?

✓ Solution:
  1. Verify ngrok tunnel is active
     → Should show: Forwarding https://... -> http://localhost:5000
  2. Verify backend is processing
     → Should show: Processing... in terminal
  3. Check GPU memory
     → Run: nvidia-smi
     → Should show GPU usage ramping up
  4. Check ngrok URL matches in console
     → F12 → Console should show correct ngrok URL
