##  COMMON ERRORS & FIXES

### **ERROR 1: "THREE is not defined"**

**Cause:** Three.js library didn't load from CDN
**Fix:**
1. Check internet connection
2. Hard reload page: **Ctrl+Shift+R**
3. Check browser console for CDN loading errors

**Verify Fix:**
Type in console: `typeof THREE`
Expected: `"object"`

---

### **ERROR 2: "threeResourceManager is not defined"**

**Cause:** Script order issue or scope problem
**Fix:**
1. Hard reload: **Ctrl+Shift+R**
2. Clear cache: Settings → Clear browsing data
3. If persists, report to me with exact line number

**Verify Fix:**
Type in console: `typeof threeResourceManager`
Expected: `"object"`

---

### **ERROR 3: "Cannot read properties of null"**

**Cause:** DOM element not found (timing issue)
**Fix:**
1. Wait for page to fully load (should be instant)
2. Check if buttons are visible on page
3. Hard reload: **Ctrl+Shift+R**

**Verify Fix:**
Type in console: `document.getElementById('gpu-test-results')`
Expected: `<div class="test-results" id="gpu-test-results"></div>`

---

### **ERROR 4: "Failed to load resource" (CDN)**

**Cause:** Internet connection or CDN down
**Fix:**
1. Check internet connection
2. Try different browser (Chrome, Firefox, Edge)
3. Check firewall/antivirus blocking CDN

**Verify Fix:**
Type in console: `typeof THREE`
Expected: `"object"` (not `"undefined"`)
