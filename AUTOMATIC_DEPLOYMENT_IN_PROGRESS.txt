IF SOMETHING GOES WRONG
═════════════════════════════════════════════════════════════════════════

Backend error:
  → Check backend terminal window
  → Look for Python/CUDA error messages
  → Verify GPU: nvidia-smi

ngrok tunnel error:
  → Check ngrok terminal window
  → Verify internet connection
  → Check firewall allows port 5000

Netlify deployment error:
  → Check main terminal output
  → Verify git repository configured
  → Try manual: netlify deploy --prod

API connection error:
  → Verify backend running
  → Verify ngrok tunnel active
  → Check netlify.toml has correct URL
  → Hard refresh browser: Ctrl+Shift+R

Cannot reach deployed site:
  → Wait for Netlify build to complete
  → Check site URL from deployment output
  → Verify no typos in URL
